## Customization

To customize this portfolio:

1. **Update Content**: Edit the text in `index.html`
2. **Change Colors**: Modify the CSS variables in the `<style>` section:
   ```css
   :root{
     --burgundy:#6B1A2A;
     --deep:#3E0D17;
     --cream:#FAF6F0;
     --gold:#C9A96E;
   }
   ```
3. **Replace Image**: Replace `aliyah.jpg` with your own photo (keep the same filename)

## GitHub Pages Setup
